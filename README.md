# SaiShivaVasam-CMS

A Content Management System (CMS) developed during my Full Stack Development Internship.

## Features

- Admin Authentication
- Read Articles
- Create Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- Search Functionality
- JSON Data Storage
- Express Routing
- EJS Templates
- Responsive UI

## Technologies

- Node.js
- Express.js
- EJS
- HTML
- CSS
- JavaScript
- JSON
